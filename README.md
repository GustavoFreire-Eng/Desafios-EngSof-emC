# Super-Trunfo-EngSof - Lóliga do super trunfo

-Menu Dinâmico Simples
Primeiro escolhe um atributo (de 1 a 6)

Depois escolhe outro atributo diferente

O programa não mostra o atributo já escolhido

-Comparações
Maior valor vence (exceto densidade)

Densidade: menor valor vence

-Resultado Final:

Soma os dois atributos de cada carta

Maior soma vence

Se empatar, mostra "EMPATE!"

Explicação Final: o jogo irá comparar os atributos escolhidos entre as cartas de acordo com o que você escolheu, a carta com os maiores valores exceto a densidade, irá vencer.
